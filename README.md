# Development

Install Elm and Webpack:

`npm install -g elm webpack`

Then install required packages:

`npm install` + `elm-install`

Run with:

`npm start`

If you want to build / minify, use:

`npm run build`

Then take `build/Main.min.js` from `/build` directory.
